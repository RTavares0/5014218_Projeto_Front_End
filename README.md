## 📱 O que o sistema faz?
Mesmo sendo uma página simples e sem banco de dados real, ela simula perfeitamente o funcionamento de um sistema de lanchonete de verdade:

* **Escolha de Loja:** O cardápio muda sozinho dependendo da cidade que você escolhe (Recife, Salvador ou Fortaleza).
* **Carrinho de Compras:** Você pode adicionar os pratos (como tapioca e cuscuz) e ele calcula o valor total na hora.
* **Pontos de Desconto (Fidelidade):** Tem um botão para simular o uso de pontos acumulados para ganhar R$ 5,00 de desconto.
* **Simulador de Pagamento:** Mostra o que acontece quando o pagamento em cartão dá certo ou quando é recusado pelo banco.
* **Acompanhamento:** Após pagar, a tela mostra o status do pedido mudando (Ex: "Na Cozinha" e "Pronto para Retirada").
* **Aviso de Privacidade (LGPD):** Tem uma barra amigável que avisa ao usuário sobre o uso seguro dos seus dados.

## 💻 Tecnologias usadas
O projeto foi feito usando apenas o básico e essencial da programação web:
* **HTML:** Para montar a estrutura e os textos da página.
* **CSS:** Para deixar o visual bonito e adaptável (funciona bem no celular, computador ou totens de autoatendimento).
* **JavaScript:** Para fazer toda a mágica dos botões, carrinho e simulações funcionarem de verdade.
